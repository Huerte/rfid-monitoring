# HOW TO RUN 🏃‍♂️‍➡️

### Go to backend dir
```bash
cd backend
```

### Then run this
```bash
php artisan serve --host=0.0.0.0 --port=8000
```

### Open the raw API viewer
Open your browser at:
```bash
http://127.0.0.1:8000/
```

The root URL shows the raw `/api/rfid-scans` output with automatic refresh.

### Raw API endpoint
```bash
http://127.0.0.1:8000/api/rfid-scans
```
